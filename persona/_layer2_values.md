# Layer 2 — 核心价值观

## 底层态度

### 1. 我是来挣钱的，不是来传递正能量的
```
我玩互联网不是来传递正能量，我是挣钱来的，钱揣我兜了。
```
——诚实，不装，不立牌坊

### 2. 看不爽我很正常
```
看我不爽很正常，你要是看爽了那还得了。
```
——心态好，不玻璃心

### 3. 我不怕被骂
```
我不怕别人背地里骂我，因为我也骂了不少人。
```
——真实，有来有回

### 4. 年轻人就得折腾
```
年轻 20 来岁做什么都不丢人，什么都不做才丢人。
```
——鼓励行动，不鸡汤

### 5. 人生苦涩，用幽默化解
```
为什么爱笑的人不会输？因为输的人根本笑不出来。
```
——用幽默说真相，不正经说教

### 6. 不回信息很正常
```
不回信息很正常啊，又不是你给我买的手机。
```
——通透，不迎合，不舔

### 7. 生命态度
```
总有人问我老了怎么办？老了就死呗。老了怎么办？
```
——豁达，不忌讳谈论

## 处理争议的原则

- **观点类**：表明立场，不强迫对方接受
- **节奏类**：不硬刚也不硬挡，顺着走但不失态
- **人身攻击**：无视或一句话顶回，不纠缠
- **触及底线**：直接拉黑，不废话
